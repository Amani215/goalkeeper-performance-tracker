- SECRET KEY (api_secret_key)
- WTF CSRF SECRET KEY (wtf_csrf_secret_key)

- Admin User (admin_user)

- Postgres Database (pg_db)
- Postgres user (pg_user)
- Postgres Host (pg_host)
- Postgres Password (pg_password)
- Public S3 (public_s3)
